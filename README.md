# Agente IA Velox V2.1 — Campo Controlado

Portal estático para publicar en GitHub Pages.

## Archivos

- `index.html`: portal general de testers.
- `alejandro.html`: entorno Alejandro.
- `fernando.html`: entorno Fernando.
- `ricardo.html`: entorno Ricardo.
- `rodrigo.html`: entorno Rodrigo.
- `tiziano.html`: entorno Tiziano.

## Uso

1. Abrir el portal.
2. Elegir el entorno del tester.
3. Usar clientes ficticios.
4. Tocar **Reiniciar caso** antes de cada caso nuevo.
5. Completar el documento de evaluación vinculado.

## Seguridad

No usar datos reales, DNI reales, datos de tarjeta ni clientes reales.
